# 🍉 Fruit Slice Game (jQuery)

A fun and interactive Fruit Slice Game inspired by Fruit Ninja, built using jQuery. Slice fruits, score points, and enjoy smooth animations!

---

## 🚀 Features

* 🎮 Interactive gameplay
* 🍎 Random fruit generation
* ✂️ Slice animations
* 🧮 Score tracking
* 🔊 Sound effects
* ⚡ Smooth and lightweight UI

---

## 🛠️ Tech Stack

* jQuery
* JavaScript
* HTML5
* CSS3
